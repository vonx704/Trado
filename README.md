TRADO is a personal trading journal app designed to help traders
log trades, track P&L, and analyze performance with discipline.

## Features
- Trade logging (BUY / SELL)
- P&L calculation
- Trade filters & search
- Analytics (Equity Curve, Monthly P&L)
- Offline-first (no data collection)
- Dark / Light / System theme

## Privacy
TRADO does not collect or share any personal data.
All data stays on your device.
